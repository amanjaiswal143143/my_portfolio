Object Following Bot

An autonomous robot capable of detecting and following specific objects using computer vision and sensors.

Python OpenCV Arduino

Equipped with real-time image processing and advanced motion control algorithms, the robot can maintain optimal distance and orientation relative to the target object. This innovative project was developed for the Makernova competition conducted under the Drishti Club of SVNIT, showcasing practical applications of robotics and AI. The system utilizes OpenCV for object detection and incorporates PID control for smooth navigation and precise movements.
